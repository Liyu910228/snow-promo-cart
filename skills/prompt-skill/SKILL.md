# 提示词 Skill

## 使用流程

1. 明确提示词用途、输入、输出和适用页面或模块。
2. 优先复用 `references/prompt-library.md` 中已有模板。
3. 新增模板时写清触发场景、变量占位、输出格式和注意事项。
4. 对外回复时给出可直接复制使用的提示词，并说明保存位置。
